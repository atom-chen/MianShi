1.C/C++采用静态编译模型，程序运行时，结构大小会在编译后确定，所以如果在结构体里定义了自身的结构体变量，编译器就会无法确定结构的大小，从而出错。

2.类递归的经典案例--两个类相互包含引用的问题
递归定义，A类定义时使用B类变量，B类定义时，使用A类变量/或者B类定义时使用C类变量
但是一直不能确定类所占字节大小。

3.类递归的实现
a.前向声明的实现
class B;
class A{
   B* num;
}
class B{
   A num;
}

b.friend声明

class A{
   friend class B;
   B* num;
}
class B{
   A num;
}

不管是哪种实现方式，有一点原则是不变的：即最多有一个类可以定义实例。
